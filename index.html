What I have learnt

From week one of the webdevelopment course I am taking..

  • HTML provides the structure and content of the webpage(s)
    • There are many different HTML tags. MDN is a good resource to find information on the best tags to use.
    • Your HTML should be semantic, this gives meaning to your page(s). Even though you can not see them on the page(s) it helps web crawlers and screen readers etc. as these read them.
  • CSS gives the webpage(s) their style. MDN is a good resource to find information about the choices available.
  • GIT is version control, that manages files on the local computer
  • github is a cloud platform where you can store files remotely. This allows...
    • you to not have to worry about loosing them if your computer breaks or gets stolen.
    • You can collaborate with other developers
    • You can contribute to projects i.e. opensource
